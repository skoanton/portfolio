# Portfolio Webpage

This project is a showcase of my front-end development skills, highlighting various projects I've worked on.

## Installation

1. Install dependencies:

```
npm install
```

2. Run the project:

```
npm run dev
```

## Technologies Used

- Typescript (with React.js)
- Tailwind
- React Router
